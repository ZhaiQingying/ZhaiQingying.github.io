非线性规划fmincon函数的运用

[x,y]=fmincon(fun,x0,a,b,aeq,beq,lb,ub,nonlcon,options)
  • fun为目标函数
  • x0为函数初始值
  • nonlcon为非线性函数构成的函数组
    fun和nonlcon里的方程组用m文件定义

  • options定义了优化参数

    min F(x) subject to: Ax <= b, Aeqx = beq 线性约束条件

    c(x) <= 0, ceq(x) = 0  非线性约束条件

    lb<= x <= ub 约束边界

二次规划

x=quadprog(h,f,a,b,aeq,beq,lb,ub,x0,options)
