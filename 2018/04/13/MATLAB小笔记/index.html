MATLAB小笔记

输入矩阵有以下几种不同方法：

  • 直接输入组成矩阵的元素。
  • 从外部文件引入矩阵。
  • 利用自带的函数构成矩阵。

  • 利用在*.M文件内自定义的函数来构成矩阵。

    直接输入

  • 用空格或者逗号来区分一行里不同的元素。

  • 用分号’；’来区分不同的行。

  • 用方括号来括住全体元素。

    矩阵行或列求和

  • sum(A):相应列的元素的和

    矩阵的转置

  • A’

    求对角线元素

  • diag(A)

    求左右调转的矩阵

  • fliplr(A)

    矩阵的元素的表达

  • A(i,j)
  • A(k)
  • 输入一个数值到矩阵以外的位置，矩阵的大小就会变化来容纳新的数据其余的为零

    冒号算子

  • i:j表示整数
  • i:j:k表示间隔为j的一组数
  • 用来表示矩阵的一部分egA(i:j) A(i:j,k)
  • ：行列的全体元素,end来表示最后一行或者最后一列的全体元素
  • A = B(:,[1 3 2 4])表示对于矩阵B的每一行，将其列的顺序按照‘1，3，2，4’重新编排

魔方矩阵 magic(n)

生成一个n*n的矩阵，矩阵元素是由整数1到n^2组成的并且任何行任何列的和都相等，阶数n必须是大于等于3的标量。

函数

查询初等函数列表,高等函数列表 ，矩阵函数列表

help elfun
help specfun
help elmat

内置常数

函数名 数值
pi 3.14159265…
i 虚数单位, 根号-1
realmin 最小浮点数，2-1022
realmax 最大浮点数，(2-)21023
Inf 无穷大
NaN 不定数

无穷大是通过非零数除以零或者良定义运算溢出（如：超过realmax）产生的;NaN不定数通常由0/0运算、Inf/Inf等良定义的运算得出。

函数名赋予任意新的变量，将会覆盖掉这些函数。原函数可以用下面的语句恢复：clear eps

创建图像

plot(x,y)x是向量，y是x的函数
x = 0:pi/100:2*pi;
y = sin(x);
plot(x,y)

添加标签和标题

xlabel(‘’)
title(‘‘,’‘,size)

多个函数关系

一个函数作图命令plot使不同的(x-y)变元函数生成不同的函数图象

自定义颜色，线型和标记

plot(x,y,’color_style_marker’)

  • 颜色字符串有’c’, ‘m’, ‘y’, ‘r’, ‘g’, ‘b’, ‘w’,和’k’。分别表示青，红紫，黄，红，绿，白和黑。
  • 线型字符串有：’-‘ 为实线, ‘–’ 为虚线, ‘:’ 为点线, ‘-.’ 为点虚线, 及’none’ 表示不用线型。
  • 标记形式有’+’, ‘o’, ‘*’,和’x’ ，填入’s’ 代表正方形, ‘d’ 代表菱形, ‘^’ 为上三角形, ‘v’ 为下三角形, ‘>’ 为右三角形, ‘<’ 为左三角形, ‘p’ 为五角星形, ‘h’ 为六角星形, none 为不用标记。

操作跟着它前面的一组因变量自变量，即plot(xfit,yfit,x，y’y‘)是对x，y加标记*

  • 只画出标记型plot(x,y,’ks’)
  • 每个数值上都有标记，并且连线plot(x,y,’r:+’)
  • 在更少的点上标记plot(x1,sin(x1),’r:’,x2,sin(x2),’r+’)

  • 冒号算子无法对复数使用，对复数的作图：

    t = 0:pi/10:2pi;
    plot(exp(i    t),’-o’)

  • 使x，y轴单位长度等长

    axis equal

在现有图中添加图

  • 原图还在

    hold on

  • 原图覆盖
    hold off

要使已有的窗口成为当前窗口

figure(n)

要打开一个新窗口并使它成为当前窗口

figure

在同一窗口中作多幅图

subplot(m,n,p)

把figure窗口分成m*n个子区域及选择第p个区域为当前图

设置轴的范围

  • 二维中

    axis([xmin xmax ymin ymax])

  • 三维中

    axis([xmin xmax ymin ymax zmin zmax])

  • MATLAB重新自动选择范围

    axis auto

设定纵横比

  • xy轴等长

    axis square

  • xy轴单位长度相等

    axis equal

  • 返回默认模式中定义的缩放比例

    axis auto normal

设定轴和网格显示或隐藏

axis on/off
grid on/off

标签标题及文本设置

  • 用xlabel, ylabel, 及zlabel 命令添加x-,y-,z-等标签。
  • 用title 命令在图象顶部加标题，
  • 用text 函数在图象中任何部位添加文本

脚本与函数

  • 在MATLAB中有两种M-文件:

脚本

不需要在其中输入参数或返回输出结果。可以在工作空间运行数据。

函数

需要给定输入参数和返回输出结果。函数文件内定义的变量仅在函数文件内部起作用。

全局变量

一个函数共用一个简单的变量，简单的处理方法就是把这个变量在所有函数中定义为global全局变量。
