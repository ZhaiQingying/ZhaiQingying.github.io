用putty快捷方式免密远程访问树莓派

发表于 2018-04-07 | 分类于 树莓派

创建公钥私钥

命令

ssh-keygen -t rsa -p""

公钥放在树莓派

命令

cat /home/pi/.ssh/id_rsa.pub>>/home/pi/.ssh/authorized_keys

私钥放在主机

windows上下载winscp去把私钥拷贝到电脑上

转换私钥

用puttygen转换保存为pi.ppk

创建快捷方式

键入位置为“putty.exe 的位置” -i “pi.ppk的位置”pi@ip地址

更改图标

我用的ico在线转换网站https://www.jinaconvert.com/cn/convert-to-ico.php

可能会有这种情况

树莓派终端输入命令sudo service ssh restart
