树莓派简单指令

发表于 2018-04-06 | 分类于 树莓派

关机：

sudo poweroff

强制关机：

sudo poweroff -f

显示ip：

sudo /etc/rc.local

重启

sudo reboot

更改密码

passwd

更新安装卸载软件

更新软件

sudo apt-get update

安装新软件包

sudo apt-get install 软件包名

卸载软件包

不删配置文件

sudo apt-get remove 软件包名

删配置文件

sudo apt-get --purge remove 软件包名
