randn（）用法

  • Y = randn（n）
    返回一个n*n的随机项的矩阵。如果n不是个数量，将返回错误信息。
  • Y = randn(m,n) 或 Y = randn([m n])
    返回一个m*n的随机项矩阵。
  • Y = randn(m,n,p,…) 或 Y = randn([m n p…])
    产生随机数组。
  • Y = randn(size(A))
    返回一个和A有同样维数大小的随机数组。
  • randn
    返回一个每次都变化的数量。

MATLAB小笔记

发表于 2018-04-13

输入矩阵有以下几种不同方法：

  • 直接输入组成矩阵的元素。
  • 从外部文件引入矩阵。
  • 利用自带的函数构成矩阵。

  • 利用在*.M文件内自定义的函数来构成矩阵。

    直接输入

  • 用空格或者逗号来区分一行里不同的元素。

  • 用分号’；’来区分不同的行。

  • 用方括号来括住全体元素。

    矩阵行或列求和

  • sum(A):相应列的元素的和

    矩阵的转置

  • A’

    求对角线元素

  • diag(A)

    求左右调转的矩阵

  • fliplr(A)

    矩阵的元素的表达

  • A(i,j)
  • A(k)
  • 输入一个数值到矩阵以外的位置，矩阵的大小就会变化来容纳新的数据其余的为零

    冒号算子

  • i:j表示整数
  • i:j:k表示间隔为j的一组数
  • 用来表示矩阵的一部分egA(i:j) A(i:j,k)
  • ：行列的全体元素,end来表示最后一行或者最后一列的全体元素
  • A = B(:,[1 3 2 4])表示对于矩阵B的每一行，将其列的顺序按照‘1，3，2，4’重新编排

魔方矩阵 magic(n)

生成一个n*n的矩阵，矩阵元素是由整数1到n^2组成的并且任何行任何列的和都相等，阶数n必须是大于等于3的标量。

自定义颜色，线型和标记

plot(x,y,’color_style_marker’)

  • 颜色字符串有’c’, ‘m’, ‘y’, ‘r’, ‘g’, ‘b’, ‘w’,和’k’。分别表示青，红紫，黄，红，绿，白和黑。
  • 线型字符串有：’-‘ 为实线, ‘–’ 为虚线, ‘:’ 为点线, ‘-.’ 为点虚线, 及’none’ 表示不用线型。
  • 标记形式有’+’, ‘o’, ‘*’,和’x’ ，填入’s’ 代表正方形, ‘d’ 代表菱形, ‘^’ 为上三角形, ‘v’ 为下三角形, ‘>’ 为右三角形, ‘<’ 为左三角形, ‘p’ 为五角星形, ‘h’ 为六角星形, none 为不用标记。

操作跟着它前面的一组因变量自变量，即plot(xfit,yfit,x，y’y‘)是对x，y加标记*

  • 只画出标记型plot(x,y,’ks’)
  • 每个数值上都有标记，并且连线plot(x,y,’r:+’)
  • 在更少的点上标记plot(x1,sin(x1),’r:’,x2,sin(x2),’r+’)

  • 冒号算子无法对复数使用，对复数的作图：

    t = 0:pi/10:2pi;
    plot(exp(i    t),’-o’)

  • 使x，y轴单位长度等长

  • 在MATLAB中有两种M-文件:
